README|====================================================================|20 March 2024

The src/ directory contains files from meworking through a series of CMSSW tutorial on
accessing particle track and vertex information in AOD file formats. They need to be run
from the src/ directory of a CMSSW_10_6_18/ working area with CMS environment set up.

Tracking tutorial with CMSSW environmentand working area setup instructions: 
https://twiki.cern.ch/twiki/bin/view/CMSPublic/WorkBookTrackAnalysis

Vertexing tutorial: https://twiki.cern.ch/twiki/bin/view/CMSPublic/WorkBookVertexReco
